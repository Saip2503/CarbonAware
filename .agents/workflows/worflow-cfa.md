# Project Workflow

## 1. Project Initialization & Development
* Create a new repository on GitHub.
* Ensure the repository is set to public.
* Open your AI platform.
* Clone your repository inside the AI platform.
* Build your solution through prompting and coding.
* Regularly commit and push your progress.
* Keep all work strictly within a single branch.

## 2. Challenge Execution
* Choose one of the provided challenge verticals.
* Design your solution and logic around that specific persona.

## 3. Submission Requirements
* Submit a public GitHub repository link.
* Ensure the complete project code is located inside the repository.
* Include a `README.md` explaining your chosen vertical, your approach and logic, how the solution works, and any assumptions made.
